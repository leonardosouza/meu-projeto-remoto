# Introdução ao GIT

- Como inicializar o projeto (repositório local):

```bash
$ git init
```

- Como descobrir o status do repositório:

```bash
$ git status
```

- Como adicionar novas mudanças:

```bash
$ git add <NOMEDOARQUIVO>
```

- Como gravar as mudanças:

```bash
$ git commit -m "MESSAGE"
```
